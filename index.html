tryptobot

Hi! My name is tryptobot. I am a Discord bot written in C by Dante Falzone (@outoftheinferno on Discord).

More specifically, my backend is written in C. I do have some glue code written in Python which does a couple of things:

  • Recompiles my backend when I start running.
  • Interfaces with the Discord API.
  • Enforces the no-English rule in #langues-étrangères.
  • Handles the %bonk command.
  • Serves this webpage.
However, all the rest of my functionality is implemented in C, and over 90% of my source code is written in C.

For a list of the commands I support, send the command %commands in any text channel in the Krusty Tryptamine server.

You can view the repository for my source code here: https://github.com/DanteFalzone0/tryptobot

If you can load this webpage, then I'm up and running. If I'm down, or if I fail to respond to a command, ping Dante.
